Graph Isomorphism

Two graphs are isomorphic if there exists a bijection between their node sets that preserves all edges. Determining isomorphism is one of the most famous open problems in computational complexity.

What is Graph Isomorphism?

Formally, graphs G = (V, E) and H = (V', E') are isomorphic (G ≅ H) if there exists a bijection φ: V → V' such that (u, v) ∈ E if and only if (φ(u), φ(v)) ∈ E'. In other words, they are the same graph drawn differently — only node labels differ.

Isomorphism is crucial in chemistry (detecting identical molecules), network analysis (identifying structurally equivalent systems), and computer science (pattern matching in graphs).

Computational Complexity

Graph Isomorphism (GI) is one of the few natural problems known to be in NP but not known to be either NP-complete or in P. This unique status has made it a landmark problem in complexity theory.

In 2015, László Babai announced a quasipolynomial-time algorithm (n^polylog(n)), a breakthrough after decades where the best known bound was subexponential. Practical solvers like nauty, Traces, and bliss run extremely fast on most real-world graphs.

P
Polynomial-time solvable special cases

Trees, planar graphs, graphs of bounded treewidth, interval graphs, and graphs of bounded degree all admit polynomial-time isomorphism algorithms.

GI
General case — quasipolynomial

Babai's algorithm runs in exp(O(log n)³) time. The best practical solvers (nauty, bliss) are often polynomial in practice despite worst-case exponential bounds.

NP
Subgraph isomorphism — NP-complete

Asking whether G contains a subgraph isomorphic to H is NP-complete (generalises clique, Hamiltonian path, and other NP-hard problems).

Necessary Conditions (Invariants)

Before running WL or exact solvers, quick invariant checks can rule out isomorphism cheaply. All of the following must match for isomorphic graphs:

  • Same number of nodes and edges
  • Same degree sequence (sorted list of node degrees)
  • Same number of connected components
  • Same clustering coefficient distribution
  • Same eigenvalue spectrum of the adjacency matrix (spectral equivalence)
  • Same k-WL certificate for all k (necessary but not sufficient)

The analyzer checks degree sequence automatically. A mismatch is a conclusive certificate of non-isomorphism.

WL as an Isomorphism Test

The Weisfeiler–Lehman test is a sound but incomplete isomorphism test:

  • Sound: if k-WL says "different", the graphs are definitely non-isomorphic.
  • Incomplete: if k-WL says "same", the graphs may or may not be isomorphic — WL cannot distinguish certain pairs (e.g. the Cai–Fürer–Immerman (CFI) graph families specifically constructed to fool k-WL).

Increasing k raises the power: 3-WL already handles most practical graphs encountered in chemistry and network science. For complete isomorphism testing, use exact solvers like nauty.
